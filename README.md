# ButaiKirin Tap

## How do I install these casks?

`brew install --cask butaikirin/tap/<cask>`

Or `brew tap butaikirin/tap` and then `brew install --cask <cask>`.

Or, in a `brew bundle` `Brewfile`:

```ruby
tap "butaikirin/tap"
cask "<cask>"
```

## Available casks

- `jiba`
